================================================================================
FLUJO DE DATOS
================================================================================

1. INGESTA:
   app_16.py → API Lichess → Descarga partidas PGN

2. ANÁLISIS TEÓRICO:
   theory_db.pkl (327K posiciones) → Identifica fin de teoría

3. ANÁLISIS ENGINE:
   Stockfish 16 → Evalúa 12 jugadas post-teoría → Calcula ACL

4. FEATURE ENGINEERING:
   orquestador.py → game_prep_score + game_risk_index

5. CLUSTERING:
   km_apertura_pura.pkl → Clasifica en 3 niveles (Dominio/Desarrollo/Sin Base)

6. RECOMENDACIÓN:
   chess_resources_v3.csv → Sistema de 3 prioridades → Recursos personalizados

7. OUTPUT:
   Dashboard + Plan de Estudio + Blunders (exportación a Lichess Studies)
