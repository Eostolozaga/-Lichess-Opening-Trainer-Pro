## Data Collection Methodology

Resource metadata in this project was curated through:

1. **Public API Endpoints**
   - YouTube Data API v3 (with appropriate API key and quota management)
   - Rate limiting and caching in compliance with API Terms of Service

2. **Public Web Catalogs**
   - Metadata extraction from publicly accessible course catalogs
   - Respecting robots.txt directives of all scraped sites
   - No circumvention of access controls or paywalls

3. **Community Contributions**
   - Manual curation of resources shared in public chess communities
   - References to freely available educational content

4. **What We Do NOT Do**
   - Download or store proprietary course content (videos, PDFs, PGNs)
   - Reproduce course descriptions or copyrighted text
   - Bypass authentication or access controls
   - Store user credentials or payment information
   - Claim ownership of any third-party intellectual property
